# 小学生听写练习系统 - 规格说明书

## 1. 项目概述

- **项目名称**：小学生听写练习系统
- **目标用户**：上海小学生家长（孩子二年级）
- **教材版本**：语文=统编版，英语=沪教牛津版
- **技术选型**：单页 Web 应用（HTML + CSS + JS，无框架），语音合成使用 Web Speech API，数据存储使用 localStorage

## 2. 功能模块

### 2.1 听写模式（核心）
选择教材 → 选择单元范围 → 设置听写参数 → 开始听写 → 逐词播放 → 家长标记对错 → 结果展示

**参数设置**：
- 每词播放次数：1~3次，默认2次
- 间隔停顿：1~10秒，默认3秒
- 播放内容选项：仅词语 / 词语+拼音 / 词语+拼音+组词

### 2.2 复习模式
针对错题本中的词语进行再次听写，支持随机打乱顺序。

### 2.3 打印模式
生成可打印的听写卷（空白卷 + 带答案卷），A4 格式，适合家长手写或孩子默写。

### 2.4 错题本
自动记录听写过程中标记为"错误"的词语，支持查看、删除、导出。

### 2.5 历史记录
记录每次听写的日期、教材范围、对错数量。

## 6. UI 页面划分

1. **首页**：选择模式（听写 / 复习 / 打印 / 历史）
2. **听写设置页**：选择教材 → 选择单元 → 设置参数
3. **听写进行页**：当前词语 + 进度条 + 播放按钮 + 对错按钮
4. **结果页**：本次听写统计 + 错误列表 + 操作按钮
5. **错题本页**：错误词语列表，支持逐个复习
6. **历史页**：历次听写记录列表
7. **打印页**：选择范围 → 生成空白卷/答案卷 → 打印

## 7. 数据持久化（localStorage）

- `wronglist`：错误词语数组，含教材来源和日期
- `history`：听写历史记录数组
- `settings`：用户设置的默认参数

## 8. 设计风格

- 简洁亲子风格，适合家长在手机/平板上使用
- 主色调：柔和蓝色 (#4A90D9)
- 字体大小：适合在嘈杂环境中观看（大字、醒目）
- 操作按钮：大尺寸，方便点击
- 整体风格参考：轻量级工具型应用

## 9. 移动端 UI 规范（第一阶段补充）

本节规定所有移动端（手机、平板）的具体实现要求，CodeBuddy 在第一阶段开发时必须遵守。

### 9.1 触控体验
- **最小触控区域**：所有可点击元素高度 ≥ 44px，宽度 ≥ 44px
- **按钮内边距**：主按钮（正确/错误）padding ≥ 16px，字号 ≥ 20px
- **消除点击延迟**：`touch-action: manipulation`（已实现）
- **点击反馈**：按下时 scale(0.95~0.98)，时长 100ms
- **禁止文字选中**：所有按钮和交互元素加 `-webkit-user-select: none; user-select: none`
- **禁止图片拖拽**：`img { pointer-events: none; }` 防止图片被长按拖拽

### 9.2 iOS 适配
- **安全区域**：使用 `env(safe-area-inset-bottom)` 和 `env(safe-area-inset-top)` 适配刘海屏和圆角屏
- **页面底部留白**：`padding-bottom: calc(16px + env(safe-area-inset-bottom, 0px))`
- **字体**：优先使用系统字体栈 `-apple-system, "PingFang SC", "Microsoft YaHei", sans-serif`
- **禁止默认触摸高亮**：`-webkit-tap-highlight-color: transparent`
- **弹性滚动**：`body { overscroll-behavior: none; }` 防止下拉刷新冲突

### 9.3 响应式断点
```
超小屏手机：< 360px     → 字号和间距进一步缩小
普通手机：  360px~600px → 默认样式
平板/大屏：> 600px     → max-width:560px 居中，字号适度放大
横屏：     orientation: landscape → 纵向布局改为紧凑横向布局
```

### 9.4 横屏适配要求
- 首页按钮从纵向堆叠改为横向排列
- 听写进行页：进度条缩小，主要区域留给对错按钮
- 结果页：分数区域缩小，统计信息横向排列
- 整体 padding 收紧

### 9.5 字号规范
```
标题/分数：≥ 36px（横屏 ≥ 28px）
主按钮文字：≥ 20px（横屏 ≥ 18px）
正文/标签：≥ 14px
辅助文字： ≥ 12px
```

### 9.6 颜色对比度
- 所有文字与背景的对比度 ≥ 4.5:1（WCAG AA）
- 正确按钮：绿色 (#4CAF50)，文字白色
- 错误按钮：红色 (#E53935)，文字白色
- 主色调：蓝色 (#4A90D9)，文字白色或深蓝
