# 小学生每日练习系统 — 功能规格文档 v6

## 文档信息
- **创建时间**: 2026-05-07
- **基于版本**: v5
- **目标年级**: 小学二年级

## 一、开发背景

v5版本已完成打印预览修复和基础多类型词库支持。v6版本重点开发：

1. **英语多类型词库** - 支持单词/短语两种输入
2. **数学自动生成系统** - 口算/竖式/应用题自动出题
3. **A4三段式练习纸** - 语文听写(横线) + 英语听写(横线) + 数学题目(可见)

---

## 二、英语多类型词库支持

### 2.1 类型定义

| 类型 | vocabType | 输入格式 | 书写横线 | 示例 |
|------|-----------|----------|----------|------|
| 单词 | `word` | 单词+释义 | 中等(约4-6em) | apple 苹果 |
| 短语 | `phrase` | 短语+释义 | 长(占满一行) | good morning 早上好 |

### 2.2 数据结构

```javascript
// 英语单词
{ text: "apple", pinyin: "", meaning: "苹果", vocabType: "word" }

// 英语短语
{ text: "good morning", pinyin: "", meaning: "早上好", vocabType: "phrase" }
```

### 2.3 批量导入解析规则

英语短语/单词的解析，以**第一个中文字符**为分界点：
- `apple 苹果` → text:"apple", meaning:"苹果"
- `good morning 早上好` → text:"good morning", meaning:"早上好"

正则: `/^([a-zA-Z\s]+)\s+([\u4e00-\u9fa5].*)$/`

## 三、数学自动生成系统

### 3.1 知识点类型

| 知识点type | 说明 | 生成规则 | 适用年级 |
|------------|------|----------|----------|
| `addition_1digit` | 一位数加法 | a,b∈[1,9] | 一年级 |
| `addition_2digit` | 两位数加法 | a,b∈[10,99], 结果≤100 | 二年级 |
| `subtraction_1digit` | 一位数减法 | a>b, a,b∈[1,9] | 一年级 |
| `subtraction_2digit` | 两位数减法 | a>b∈[10,99] | 二年级 |
| `multiplication_99` | 乘法口诀 | a,b∈[1,9] | 二年级 |
| `division_simple` | 简单除法 | 先乘法a×b=c, 再c÷a= | 二年级 |
| `vertical_addition` | 竖式加法 | 两位数+两位数,需进位 | 二年级 |
| `vertical_subtraction` | 竖式减法 | 两位数-两位数,需退位 | 二年级 |
| `word_problem` | 应用题 | 根据模板生成 | 二年级 |

### 3.2 应用题模板

```javascript
const WORD_PROBLEM_TEMPLATES = [
  {
    type: 'addition',
    template: '小明有{a}个苹果，又买了{b}个，现在一共有多少个苹果？',
    answer: (a, b) => a + b
  },
  {
    type: 'subtraction',
    template: '篮子里有{a}个橘子，吃掉了{b}个，还剩多少个橘子？',
    answer: (a, b) => a - b
  },
  {
    type: 'multiplication',
    template: '每盒有{a}颗糖果，{b}盒一共有多少颗糖果？',
    answer: (a, b) => a * b
  }
];
```

### 3.3 数学题目数据结构

```javascript
// 口算题
{ text: "23+45=", type: "oral", answer: "68" }

// 竖式题
{ text: "  56\n+ 27\n────", type: "vertical", answer: "83" }

// 应用题
{ text: "小明有23个苹果，又买了15个，现在一共有多少个苹果？", type: "word", answer: "38" }
```

## 四、A4三段式练习纸

### 4.1 布局设计

```
┌────────────────────────────────────────┐
│           每日练习 - 2026年5月7日        │
│              二年级 张三                │
├────────────────────────────────────────┤
│ 【语文听写】                            │
│  1. ____________  2. ____________      │
│  3. ____________  4. ____________      │
│  （词语：短横线，古诗：长横线）          │
├────────────────────────────────────────┤
│ 【英语听写】                            │
│  1. ________________  2. __________    │
│  3. ________________  4. __________    │
│  （单词：中等横线，短语：长横线）        │
├────────────────────────────────────────┤
│ 【数学练习】                            │
│  1. 23 + 45 = ______                   │
│  2. 56 ÷ 8 = ______                    │
│  3. 7 × 8 = ______                     │
│  （题目可见，答案横线）                  │
└────────────────────────────────────────┘
```

### 4.2 打印规格

- **纸张**: A4 (210mm × 297mm)
- **边距**: 上下15mm，左右12mm
- **字号**: 正文12pt，标题14pt
- **行距**: 1.8倍

### 4.3 横线长度规则

| 内容类型 | 横线长度 | 说明 |
|----------|----------|------|
| 语文单字/词语 | 3-5em | 按字数调整 |
| 语文短语/古诗 | 100% | 占满一行 |
| 英语单词 | 4-6em | 按字母数调整 |
| 英语短语 | 100% | 占满一行 |
| 数学答案 | 3em | 固定长度 |

## 五、听写播报流程

### 5.1 语文听写播报

1. 播报标题："第一题"
2. 播报词语："池塘"（重复3次）
3. 停顿3秒
4. 继续下一题

### 5.2 英语听写播报

1. 播报标题："Number one"
2. 播报单词："apple"（重复3次）
3. 停顿3秒
4. 继续下一题

### 5.3 数学题目播报

数学题目**不播报**，直接显示在A4纸上供学生作答。
