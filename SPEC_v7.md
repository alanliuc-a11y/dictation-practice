# 小学生每日练习系统 — 功能规格文档 v7

## 文档信息
- **创建时间**: 2026-05-07
- **基于版本**: v6
- **新增功能**: 错题复习模式

## 一、功能背景

当前系统已有错题本功能，但只能查看错题，无法针对错题进行专项练习。v7版本新增**错题复习模式**，让学生可以：

1. 选择错题本中的内容进行专项听写练习
2. 支持随机打乱顺序，避免机械记忆
3. 练习完成后自动更新错题掌握状态
4. 已掌握的错题可从错题本移除

## 二、错题复习模式

### 2.1 功能流程

```
首页 → 错题本 → 选择复习模式 → 开始错题听写 → 完成 → 更新错题状态
```

### 2.2 错题本页面增强

在现有错题本页面增加：
- **复习模式按钮**: 一键开始错题复习
- **选择功能**: 可选择部分错题进行复习
- **统计信息**: 显示错题总数、已掌握数、待复习数

### 2.3 错题听写流程

与普通听写流程类似，但：
1. 只播报错题内容
2. 不显示词语（保持听写性质）
3. 完成后可选择：
   - 标记为"已掌握"（从错题本移除）
   - 保留在错题本（继续复习）

### 2.4 数据结构

错题数据增加字段：
```javascript
{
  text: "池塘",
  pinyin: "chí táng",
  subject: "chinese",
  addedAt: "2026-05-01T10:00:00Z",
  reviewCount: 2,        // 复习次数
  lastReviewed: "2026-05-07T15:30:00Z",  // 最后复习时间
  mastered: false        // 是否已掌握
}
```
