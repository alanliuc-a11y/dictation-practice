# 小学生每日练习系统 — 功能规格文档 v8

## 文档信息
- **创建时间**: 2026-05-08
- **基于版本**: v7
- **新增功能**: 词库架构升级（基础词库+自定义词库分区）

## 一、词库架构设计

### 1.1 双层词库体系

```
📚 基础词库（VOCAB - 代码内置，只读）
├── 语文上册（9课，部编版二年级上册）
├── 语文下册（29课，部编版二年级下册）✨ 新增
├── 英语上册（待补充）
├── 英语下册（待补充）
└── 数学（自动生成，无需词库）

✏️ 自定义词库（CustomVocab - localStorage，可增删改）
├── 语文自定义（老师要求的组词、拓展词语、古诗词）
├── 英语自定义（老师要求的短语、句型）
└── 数学自定义（暂无）
```

### 1.2 关键区别

| 特性 | 基础词库 | 自定义词库 |
|------|----------|------------|
| 存储位置 | data.js 代码文件 | 浏览器 localStorage |
| 可编辑 | ❌ 只读 | ✅ 可增删改 |
| 学期切换 | ✅ 按学期筛选 | ❌ 始终全部显示 |
| 数据来源 | AI根据教材编写 | 家长手动输入 |
| 适用场景 | 课本生字词 | 老师拓展要求 |

### 1.3 选择页分区显示

设置页中，语文和英语各分为两个区域：
- **📖 课本词库**：根据学期下拉框切换显示
- **✏️ 自定义词库**：始终显示全部自定义词库
