## 一、后台管理系统（新增）

### 1.1 设计理念

为家长/老师提供基础词库的维护能力，无需修改代码即可管理词库内容。通过隐藏入口进入，不影响学生正常使用。

### 1.2 进入方式

| 方式 | 操作 |
|------|------|
| 隐藏入口 | 首页连续点击标题"小学生每日练习"5次（2秒内） |
| URL直入 | 浏览器地址栏添加 `#admin`，如 `http://xxx:8080/#admin` |

### 1.3 功能模块

```
后台管理页 (page-admin)
├── 学期选择器（与前台共用）
├── 📚 语文基础词库管理
│   ├── 课文列表（显示名称、词语数、类型标签）
│   ├── + 添加课文 → 课文编辑页
│   ├── 批量导入
│   └── 点击课文 → 编辑课文
├── 📖 英语基础词库管理
│   ├── 单元列表
│   └── + 添加单元 → 课文编辑页
├── 🔢 数学基础词库管理 ✨ 新增
│   ├── 知识点列表
│   └── + 添加知识点 → 数学知识点编辑页
└── 数据管理
    ├── 导出词库（JSON）
    ├── 导入词库（JSON）
    └── 重置为基础词库
```

### 1.4 课文编辑页 (page-admin-lesson-edit)

| 字段 | 说明 |
|------|------|
| 课文名称 | 文本输入，如"第10课 坐井观天" |
| 词语类型 | 下拉选择：普通词语 / 古诗词 |
| 词语列表 | 逐个添加，输入词语后按回车 |
| 拼音 | **自动生成**（依赖 pinyin-pro 库），无需手动输入 |
| 操作 | 保存课文 / 删除课文 |

### 1.5 数学知识点编辑页 (page-admin-math-edit) ✨ 新增

| 字段 | 说明 |
|------|------|
| 知识点名称 | 文本输入，如"加法练习" |
| 知识点配置 | 复选框选择题型 + 数量调节（1-20题） |
| 预览 | 点击"生成预览"查看生成的题目和答案 |
| 操作 | 保存知识点 / 删除知识点 |

支持的题型（11种）：

| 题型 | 默认数量 |
|------|----------|
| 加法（1位数） | 5 |
| 加法（2位数不进位） | 5 |
| 加法（2位数进位） | 5 |
| 减法（1位数） | 5 |
| 减法（2位数不退位） | 5 |
| 减法（2位数退位） | 5 |
| 乘法（1-9） | 5 |
| 除法（简单） | 5 |
| 应用题（加法） | 3 |
| 应用题（减法） | 3 |
| 应用题（乘法） | 3 |

### 1.6 数据持久化

- 存储位置：`localStorage`，key 为 `builtin_vocab_data`
- 无自定义数据时回退到 `data.js` 中的 VOCAB 默认数据
- 支持导出/导入 JSON，方便备份和迁移

## 二、拼音自动生成（新增）

### 2.1 技术方案

- 使用 `pinyin-pro@3.28.1` 库（CDN 加载）
- 全局变量：`pinyinPro`
- 调用方式：`pinyinPro.pinyin(text, { toneType: 'symbol' })`
- CDN 地址：`https://cdn.jsdelivr.net/npm/pinyin-pro@3.28.1/dist/index.js`

### 2.2 使用场景

| 场景 | 行为 |
|------|------|
| 后台添加课文词语 | 输入词语 → 按回车 → 自动生成带声调拼音 |
| 后台编辑课文词语 | 加载已有词语时显示已有拼音 |
| 前台词库编辑 | 批量导入时支持 `词语 拼音` 格式，拼音可选 |

### 2.3 生成示例

| 输入 | 自动生成 |
|------|----------|
| 井沿 | jǐng yán |
| 回答 | huí dá |
| 无边无际 | wú biān wú jì |
| 坐井观天 | zuò jǐng guān tiān |

## 四、前台词库管理修复

### 4.1 数学科目修复

**问题**：选择数学后，词库编辑页仍显示"课文名称"输入框和词语列表区域。

**修复**：`updateVocabInputSection(subject)` 函数在 `subject === 'math'` 时隐藏以下区域：
- 诗词输入区 (`poem-input-section`)
- 词汇导入区 (`vocab-import-section`)
- 词汇列表区 (`vocab-words-section`)
- 课文名称行 (`vocab-name`)

### 4.2 数学设置页内容列表升级

**改动**：`renderMathContentList()` 函数升级为两段式显示：

```
数学内容列表
├── 📚 后台知识点（从 BuiltinVocab 获取）
│   └── 复选框列表，显示名称和预估题目数
└── ⚡ 快速选择（默认数学类型）
    ├── 一位数加法
    ├── 一位数减法
    └── 乘法口诀
```

### 4.3 数学题目收集升级

`collectMathWords()` 函数支持两种数据源：
- `builtin:grade:key` 格式：从 BuiltinVocab 获取知识点配置，调用 MathGenerator 生成
- 默认类型格式：直接使用配置的 count 参数

## 五、语数外三科统一架构

### 5.1 各科功能对照

| 功能 | 语文 | 英语 | 数学 |
|------|------|------|------|
| 前台设置页选择 | ✅ 课本词库+自定义词库 | ✅ 课本词库+自定义词库 | ✅ 后台知识点+快速选择 |
| 后台管理 | ✅ 课文增删改 | ✅ 单元增删改 | ✅ 知识点增删改 ✨ |
| 输入方式 | 词语+自动拼音 | 单词+释义 | 知识点类型+数量 |
| 自动出题 | ❌ | ❌ | ✅ MathGenerator |
| 数据存储 | BuiltinVocab + CustomVocab | BuiltinVocab + CustomVocab | BuiltinVocab + MathGenerator |
| 听写播报 | ✅ TTS 中文语音 | ✅ TTS 英文语音 | ✅ TTS 中文读题 |

### 5.2 数据流

```
语文: VOCAB(默认) → BuiltinVocab(localStorage可编辑) → 设置页选择 → TTS播报
英语: VOCAB(默认) → BuiltinVocab(localStorage可编辑) → 设置页选择 → TTS播报
数学: 知识点配置 → MathGenerator.generate() → 实时生成 → TTS播报
自定义: CustomVocab(localStorage) → 设置页选择 → TTS播报
```
